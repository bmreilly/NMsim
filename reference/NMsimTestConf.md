# Summarize and test NMsim configuration

Summarize and test NMsim configuration

## Usage

``` r
NMsimTestConf(
  path.nonmem,
  dir.psn,
  method.execute,
  must.work = FALSE,
  system.type
)
```

## Arguments

- path.nonmem:

  See ?NMsim

- dir.psn:

  See ?NMsim

- method.execute:

  See ?NMsim

- must.work:

  Throw an error if the configuration does not seem to match system.

- system.type:

  See ?NMsim

## Value

A list with configuration values

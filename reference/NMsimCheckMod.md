# Check a simulation control streams for things that can cause trouble in NMsim

Check a simulation control streams for things that can cause trouble in
NMsim

## Usage

``` r
NMsimCheckMod(file.mod, lines)
```

## Arguments

- file.mod:

  A control stream to check

- lines:

  The control stream as text lines. Only use of of \`file.mod\` and
  \`lines\`.

# Internal function to run Nonmem on linux

Internal function to run Nonmem on linux

## Usage

``` r
NMrunLin(
  fn.mod,
  dir.mod.abs,
  exts.cp,
  meta.tables,
  path.nonmem,
  clean,
  sge,
  nc,
  pnm,
  nmfe.options,
  fun.post = NULL
)
```

## Arguments

- fn.mod:

  Just the file name, not including path
